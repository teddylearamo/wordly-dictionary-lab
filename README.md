# Wordly Dictionary App

## Description
This is a Single Page Application (SPA) that allows users to search for word definitions using a dictionary API.

## Features
- Search for a word
- Displays pronunciation, definition, and example
- Handles errors for invalid input
- Updates page dynamically (no reload)

## How to Run
1. Open index.html in your browser
2. Enter a word and click search

## Technologies Used
- HTML
- CSS
- JavaScript
- Fetch API
